# Translation app
Streamlit app to translate English to Japanese using Google Translator and Google Text-to-Speech.
The app will create an audio file to read out the translated text. 

![Alt Text](https://i.gyazo.com/6d324dd90dece6e31a93942236ab5b16.gif)

The user can also upload audio of speech which will be transcribed and translated.
